
Tour Page Redesign Remarks (Samarkand City Tour)
================================================

Reference: Elephant Jungle Sanctuary Page (GetYourGuide-style layout)
Goal: Match clean, modern, image-driven, and SEO-optimized design while maintaining branding consistency.

------------------------------------------------------------
UI / UX REMARKS
------------------------------------------------------------
1. HERO SECTION
----------------
• The hero section currently shows a large blank/gray area — main image not loading or styled with incorrect object-fit/height.
• The thumbnails are inconsistent in size and alignment.
• Compared to the reference, the hero should feature a full-width main image (3:1 or 16:9 ratio) with uniform thumbnails.

✅ FIX:
- Use CSS grid or flexbox for a two-column layout (main image left, four thumbnails right).
- Apply `aspect-ratio: 16 / 9` and `object-fit: cover` to all images.
- Preload the main image; lazy-load the rest with `loading="lazy"`.

2. TYPOGRAPHY & SPACING
------------------------
• Titles and breadcrumbs are cramped vertically.
• Add consistent vertical rhythm (`padding-block: 2rem;`).
• Reference page uses ample white space and clear section divisions.

✅ FIX:
- Increase section spacing using consistent padding variables.
- Maintain readable line height (1.4–1.6) and color contrast > 4.5:1.

3. COLOR & VISUAL HIERARCHY
----------------------------
• Header background and beige body lack separation — flat look.
• CTA button is bright but the page lacks visual depth.

✅ FIX:
- Add a soft box-shadow to the header and section cards.
- Use a darker text color (#1a1a1a) and lighter background contrast.
- Highlight key elements using accent brand color (#F4B400).

4. IMAGE QUALITY & RATIOS
--------------------------
• Some images fail to load (path or case issue).
• Aspect ratios are inconsistent, causing awkward gaps.

✅ FIX:
- Convert all images to WebP, 800×600px minimum.
- Use consistent names (lowercase, no spaces) and ensure URLs are absolute (`https://jahongirtravel.com/...`).

5. BREADCRUMBS
--------------
• Breadcrumb contrast too low; link hierarchy okay.
✅ FIX:
- Use subtle gray (#6c757d).
- Add `aria-label="breadcrumb"` for accessibility.

6. TOUR META LINE
-----------------
• Meta icons appear visually heavy and cramped.
✅ FIX:
- Replace Font Awesome icons with inline SVGs.
- Use `.meta-item { display: inline-flex; align-items: center; gap: 0.25rem; }`.

7. SECTION NAVIGATION (Overview, Highlights, etc.)
--------------------------------------------------
• Inline links jammed together, missing active states.
✅ FIX:
- Convert into a sticky sub-navigation bar with scroll spy highlighting.
- Use `<nav aria-label="Tour Sections">` for semantics.

8. CALL TO ACTION (Booking)
----------------------------
• Booking card sits below fold, not sticky.
✅ FIX:
- Add `position: sticky; top: 120px;` for desktop.
- Add mobile floating CTA with price and “Book Now” button (bottom fixed).

------------------------------------------------------------
TECHNICAL REMARKS
------------------------------------------------------------
9. IMAGE LOADING & PERFORMANCE
-------------------------------
• Hero image not preloaded → poor LCP.
• Missing width/height attributes → CLS risk.

✅ FIX:
- Preload hero image.
- Use `width`/`height` attributes + lazy loading for others.

10. FONT AWESOME & ICONS
------------------------
• Font Awesome adds ~80KB + render delay.
✅ FIX:
- Replace with inline SVGs or small local sprite.

11. JSON-LD & META TAGS
-----------------------
• Good inclusion of TouristTrip schema.
✅ FIX:
- Ensure all image URLs are absolute.
- Add `sameAs` array linking social profiles for richer SERP cards.

12. SEO & ACCESSIBILITY
-----------------------
✅ FIX:
- Add descriptive alt text for each image:
  Example: `alt="Registan Square in Samarkand – Silk Road landmark"`
- Ensure all headings follow H1 > H2 > H3 structure.
- Add skip link before nav for keyboard navigation.

------------------------------------------------------------
VISUAL STRUCTURE RECOMMENDATIONS
------------------------------------------------------------
• Use grid structure like this:

  [ Main Hero Image  |  Thumbnails (4 stack) ]
  [ Tour Title + Meta line ]
  [ Section Nav (sticky) ]
  [ Overview ]
  [ Highlights ]
  [ Includes/Excludes ]
  [ Booking Sidebar (sticky) ]
