Minimal JS (optional, enhance details + GA events)
--------------------------------------------------
// If you prefer buttons instead of <details>, use this pattern.
// With <details> above, JS is not required. Below adds analytics.
document.querySelectorAll('.accordion__item').forEach((d) => {
  d.addEventListener('toggle', () => {
    if (d.open && window.gtag) {
      gtag('event', 'footer_accordion_open', { section: d.querySelector('.accordion__summary').textContent.trim() });
    }
  });
});

Accessibility Notes
-------------------
- Use <details>/<summary> for native, keyboard-accessible accordions.
- Ensure focus states are visible on links and controls.
- Icon-only elements must include aria-label or hidden text.
- Maintain a 44x44px min tap target for links and controls.
- Color contrast ‚â• 4.5:1 (current palette meets this).

Performance
-----------
- Inline the footer CSS into the page‚Äôs main CSS bundle.
- SVG icon sprites for the social icons; avoid heavy icon fonts on mobile.
- Lazy-load locale modal JS on first interaction (click).
